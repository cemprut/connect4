# Connect 4

A terminal-based 2-player Connect 4 game built with Python and curses library. Features a colorful interface, smooth cursor movement, and classic Connect 4 gameplay for two players.

## Features

- 2-player gameplay
- Visual cursor
- Colorful interface
- Centered display
- Simple controls

## Requirements

- Python 3.6 or higher
- Windows, macOS, or Linux
- Terminal that supports colors

## Installation

1. Clone or download this repository
2. Make sure you have Python installed on your system
3. No additional packages needed - uses only standard library modules

## How to Run

```bash
python connect4.py
